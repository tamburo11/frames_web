WHAT IS FRAMES


FRAMES is a free graphical spectral processor for Ableton Live, developed with Max for Live (M4L) by Alberto Barberis and Alberto Ricca. FRAMES allows you to manipulate a sound source of an Ableton Live track using a sound synthesis technique called analysis-resynthesis with graphical interaction. The implementation of this technique is based on the amazing work by Jean-Francois Charles. For more info check this.


FRAMES allows you to create a 2D image of the spectrogram of a sound source (with FFT analysis), to manipulate it with some graphical transformations and to resynthesize it in loop and in real-time.

The loop region is in sync with the tempo and the time signature of Ableton Live. The FFT analysis can be performed with a size of 512, 1024, 2048, 4096 samples (adapting it to the type of sound material).

The possible sound-image transformations are:

  • a spectral gate effect (with two different modes that allows you to maintain only the spectrum components with a magnitude higher or lower than a threshold );

  • a spectral time stretch (changing the playback speed);

  • a spectral "time-blur" (Future) or "time-drier" (Past) (a sort of spectral reverb amount);

  • the application of a transformation matrix, that allows you to perform scaling, rotation, skewing and perspective operation on the spectrogram;

FRAMES offers a deep user interface to control the graphical transformations parameters, with immediate sonical results. Besides you can set the amount of processing with a Dry/Wet control, and also save two different presets and interpolate between them.

The result is a simple spectral processing tool that allows you to synthesize unexpected sounds, complex spectral textures and irregular rhythmic loops, allowing an innovative exploration of the spectral possibilities of a sample.

HOW TO INSTALL FRAMES


  1. Download the FRAMES folder from THIS REPOSITORY.

  2. Unzip the folder and place it in your Ableton User Library folder: Ableton/User Library/Presets/Audio Effects/Max Audio Effects;

  3. Open Ableton Live; you will find the device here: Max for Live > Max Audio Effect > FRAMES_v.1.2.2.amxd; place it in an audio track;

HOW TO USE FRAMES



  1. Send an audio to the device (for example playing a clip in the track that contains the device);

  2. Choose the FFT frame size ① (adapting it to the type of sound material: 512 samples for percussive sounds, 4096 for texture sounds);

  3. Choose the length of the sample that you want to record and process ②;

  4. Clik the Record button ③ (if you ckick on the Play button ④ you will hear the real-time resynthesis);

  5. Start your graphical manipulation:
    • using the two types of gate effect ⑤ (or bypass it clicking on pass);
    • changing the playback speed ⑥;
    • changing the amount of the "spectral reverb" ⑦;
    • changing the transformation matrix parameters being inspired by the symbols ⑧ (when everything goes crazy use the Reset button);

  6. When you find a cool configuration save it in a preset (1 or 2) and try to interpolate between presets;

  7. Set your desired output gain and Dry/Wet amount;

  8. Enjoy!
